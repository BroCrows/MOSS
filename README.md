# MOSS - MyOtakuSpreadSheet

MOSS is an anime tracking system built within Google Sheets. 
Its built with multiple spreadsheets, each managing different aspects of a tracking system.

## Structure
- **MetaData** Masterlist of all anime and their attributes
- **UserList** Tracks your watch status and ratings
- **UserData** Combines MetaData and UserList aswell as managing settings
- **Recommendations** Generates anime suggestions using user data
